Main - создает интерфейсы ввода вывода и логики (и их реализаций) и запускает игру.
GameFlow - осуществляет общий контроль игры.
Input - принимает ход игрока.
Output - вывод на консоль.
Logic - расчет логики игры.
Queue - определение очередности хода.
Matrix - определяет параметры таблицы.
Cell - клас отображающий состояние ячейки.
Board - класс отображающий состояние игрового поля.
