Main - создает интерфейсы ввода вывода и логики (и их реализаций) и запускает игру.
Play - осуществляет общий контроль игры.
Input - принимает ход игрока.
Output - вывод на консоль.
LogicInterface - расчет логики игры.
QueueInterface - определение очередности хода.
MatrixInterface - определяет параметры таблицы.
